# Typography

For Typography we create custom Tailwind classes with the plugin functionality in the tailwind.config  
all starts with prefix `mumble-font-*`

- `mumble-font-h1`
- `mumble-font-h2`
- `mumble-font-h3`
- `mumble-font-h4`
- `mumble-font-paragraph-l`
- `mumble-font-paragraph-m`
- `mumble-font-label-xl`
- `mumble-font-label-l`
- `mumble-font-label-m`
- `mumble-font-label-s`
- `mumble-font-placeholder`

more information about plugins: [https://tailwindcss.com/docs/plugins](https://tailwindcss.com/docs/plugins)
